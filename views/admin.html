Client-side data connector

If you use the Website connector
Simply use the following URL to load this connector as a custom script : 
https://<%=host%>/ship.js#<%= <%= id %> %>
ELSE
Paste this snippet in your page to get access to User data from Hull
<script src="https://<%=host%>/ship.js" type="text/javascript" async defer data-hull-id="<%= id %>" data-hull-endpoint="https://<%=host%>" charset="utf-8"></script>
